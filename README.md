# Dashboard Financiero Agroven

Simulador interactivo de viabilidad financiera para proyecto ganadero (Brahman/F1).

## Requisitos

- Python 3.8+
- Entorno virtual recomendado

## Instalación

1.  Crear entorno virtual:
    ```bash
    python3 -m venv venv
    source venv/bin/activate  # En Mac/Linux
    # venv\Scripts\activate  # En Windows
    ```

2.  Instalar dependencias:
    ```bash
    pip install -r requirements.txt
    ```

## Ejecución

Para iniciar el dashboard:

```bash
streamlit run app.py
```

El navegador debería abrirse automáticamente en `http://localhost:8501`.
